# log-analyzer
This project analyzes log files and extracts useful information using Regular Expressions and Python file handling.

## 🚀 Features
- Reads log files
- Extracts patterns using regex
- Filters important information
- Processes large text data efficiently

---

## 🧠 Concepts Used
- File Handling
- Regular Expressions (re)
- Loops and Conditions
- String Processing
- Functional Programming (optional: map/filter)

---

## 💻 Tech Stack
- Python

## 📁 How to Run
1. Place log file in project folder  
2. Run Python script  
3. View extracted results
